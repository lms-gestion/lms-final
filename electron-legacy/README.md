# LMS Gestion v0.0.1 — La Maison des Services

## Lancement rapide
1. Installer Node.js : https://nodejs.org (LTS)
2. Ouvrir CMD dans ce dossier
3. `npm install` puis `npm start`

## Compiler le .exe
`npm run build` → dossier dist/

## Google Maps (adresses auto)
Remplacer YOUR_KEY dans src/index.html ligne 8

## Données sauvegardées
%APPDATA%\lms-gestion\lms-data.json
